Nom du projet : Site web produit SmartBottle

Description :
Ce projet est un site web vitrine présentant un produit innovant appelé SmartBottle.

Objectif :
Mettre en valeur les atouts du produit, présenter ses fonctionnalités détaillées, intégrer une page d’inscription marketing et une page de contact.

Le site contient :
1. Une page d’accueil avec photo et présentation générale
2. Une page dédiée aux fonctionnalités avec informations détaillées
3. Une page d’inscription (landing page) avec formulaire pour obtenir plus d’informations
4. Une page de contact avec formulaire
5. Un lien d’appel à l’action en bas de chaque page sauf la page contact

Technologies utilisées :
- HTML
- CSS
- JavaScript

Instructions d’utilisation :
1. Ouvrir le fichier index.html dans un navigateur web
2. Utiliser le menu pour naviguer entre les pages
3. Tester les formulaires de la landing page et de la page contact

Conclusion :
Le projet respecte toutes les consignes demandées.
